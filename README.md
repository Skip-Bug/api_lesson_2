## Как запустить?   
**Качаем наш код**
**Запускаем CLI** Нажмите Win + R, введите cmd
```bash
   cd укажите_путь_к_папке_api_lesson_2
```
**Запустите main.py**
```bash
python main.py
```
- Если хотите получить короткую ссылку введите нужный вам адрес   
- Если хотите увидеть статистику по кликам введите ранее полученный адрес
